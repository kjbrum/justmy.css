# Sprinkles.css

> A starter CSS framework to make development more enjoyable.


## Documentation

http://kylebrumm.com/sprinkles.css/


## Features

- [x] Typography (Headings, paragraph, bold, italic, font-family)
- [x] Lists
- [x] Tables
- [x] Definition list (dl, dt, dd)
- [x] Blockquotes
- [ ] Code & pre
- [x] Buttons
- [ ] Forms
- [ ] Colors
- [ ] Utility classes
- [ ] Breakpoints
- [ ] Functions
- [ ] Mixins
